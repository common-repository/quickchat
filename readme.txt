Плагин для добавления бесплатного онлайн-консультанта QuickChat на ваш сайт.

== Description ==

Бесплатный онлайн-консультант QuickChat для вашего сайта, который предоставляет вам все инструменты для привлечения клиентов.

Минимальные требования для этого плагина - PHP 5.4

== Installation ==

1. Загрузите .zip архив плагина через "Плагины"-->"Добавить новый"-->"Загрузить плагин"

2. Активируйте установленный плагин.

3. Найдите установленный плагин в меню "Настройки" и вставьте в него код онлайн-консультанта.

== Frequently Asked Questions ==

= Можно ли использовать QuickChat бесплатно? =

Да, вы можете неограниченно пользоваться бесплатной версией продукта. Если вам необходим расширенный функционал, то вы можете приобрести платный пакет со скидкой.

= Чем QuickChat отличается от конкурентов? =

Вам предоставляется широкий бесплатный функционал, достаточный для полноценного консультирования клиентов.

Кроме того, для привлечения внимания к вашему сайту вы можете использовать видеозапись оператора и выгодно отличаться от своих конкурентов.

= Как я буду получать сообщения от клиентов? =

Для этого вам нужно воспользоваться операторской программой и войти в неё, используя свои регистрационные данные.

Вы можете использовать как веб-версию программы, так и установить её себе на компьютер.
